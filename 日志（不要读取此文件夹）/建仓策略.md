## 加仓触发机制：盈利里程碑

不按日历走，**按账户盈利情况触发每次操作动作**：


| 里程碑      | 触发条件                      | 动作              | 操作后目标 |
| -------- | ------------------------- | --------------- | ----- |
| **当前**   | 现在（4.87万）                 | 分配现金，建好2只仓位     | 合理持仓  |
| **里程碑1** | 总资产盈利 **+3,000**（≥51,649） | 加仓已盈利品种 / 开第3只票 | ~5.2万 |
| **里程碑2** | 总资产盈利 **+5,000**（≥56,649） | 加仓 / 视情况引入更多资金  | ~5.7万 |

| **目标**   | 总资产达到 **10万**             | 达成 ✅            | —     |


**逻辑**：赚3,000证明一次判断力 → 再放大仓位。赚不到就不加，避免越陷越深。

## 熔断机制：亏了就刹车


| 情况              | 动作             |
| --------------- | -------------- |
| 单只股票**亏损>8%**   | 减仓到半仓，评估是否止损   |
| 单只股票**亏损>12%**  | 强制止损           |
| 总账户从高点**回撤>8%** | 减仓到半仓，暂停加仓直到回本 |
| 连续2笔交易亏损        | 复盘后再操作         |


**赚了加油门，亏了踩刹车**——这是整个策略的灵魂。
