# 下行期（大盘趋势走弱或震荡时使用）

## 选股——逆势资金流入方向
找今日净流入的股票，股价涨幅不大，或者微跌的股票。在逆势吸筹或者大盘企稳后大概率领涨的股票。需要结合技术指标和基本面，热点等综合判断。并给出逻辑支持。只看主板和创业板。目标是分析检索出明日可能大涨的股票。

## 选股——逆势抗跌+资金介入
在大盘近3-5日下跌的背景下，找出同期逆势微涨或横盘不跌、且资金持续净流入的个股。机构参与度>40%为加分项但非硬性门槛，重点看"大盘暴跌日走平或微涨+主力净流入"的组合信号。同板块出现多只候选时要横向对比，优先推荐涨幅较小（还没启动）的那只。

## 选股——缩量洗盘信号
查找近5日有过单日涨幅超5%、但随后2-3天缩量回调（成交量缩至放量日的一半以下）的个股，判断是否是主力洗盘结束的信号

## 波段跟踪——已卖出股票的回调买入机会

**核心教训**：卖出一只股票后不要彻底放手，尤其是在上升趋势中因短期回调而卖出的股票。回调往往是重新上车的机会。

**触发条件**：检查我近5个交易日内卖出的股票（参考持仓记录），是否出现以下回调买入信号：

1. **缩量回调**：卖出后连续2-3日成交额逐步萎缩（每日递减），说明卖压衰竭而非恐慌出逃
2. **主力资金回流**：回调期间主力从大幅净流出转为净流入（关键拐点），说明聪明资金已在抄底
3. **技术支撑有效**：股价回落到BOLL中轨、MA20或前期平台附近获得支撑，未破位
4. **RSI回归中性**：RSI从超买区（>70）回落到40-60中性区间，短期超买已消化
5. **上升趋势未破**：MA5/MA10仍向上发散，整体上升趋势保持完好

**实战参考（阳光电源 2026.03）**：
- 3/17卖出后，3/17→3/18→3/19成交额124亿→105亿→97亿，逐步缩量
- 3/17主力净流出-17.53亿，但3/19主力已转为净流入+2.23亿（资金拐点）
- 股价回落到163-165获支撑，RSI从86回到53
- 3/20强势反弹+6.53%，如果3/19尾盘或3/20早盘买回可吃到整段利润

**操作建议**：每天盘前/盘后对近期卖出的股票跑一遍上述信号检查，满足3条以上可考虑回补波段仓位。

# 分析规范（每次出报告时遵守）

1. **风险分级**：多个超买信号叠加时标注⚠️高危并建议仓位减半，模型上涨概率<45%时显性提示偏空
2. **资金趋势**：资金分析必须同时展示绝对值和趋势方向，连续递减时标注"动能衰竭"
3. **报告一致性**：前序已降级的标的，新报告必须引用前序结论并说明是否变更
4. **高波动止损**：换手>8%的股补充跳空预案，给出开盘低于止损价时的应对方案
5. **同板块对比**：候选列表中同板块个股要横向比较，不能分散在不同策略里各自评分
